## ⚡ Schnellstart in 3 Schritten

### 1️⃣ Android Studio öffnen
- Starten Sie Android Studio
- Klicken Sie auf "Open"
- Wählen Sie diesen Ordner

### 2️⃣ Warten
- Android Studio richtet alles automatisch ein
- Das dauert 5-10 Minuten beim ersten Mal
- Warten Sie auf "Gradle sync finished"

### 3️⃣ Run klicken
- Klicken Sie auf den grünen Play-Button ▶️
- Wählen Sie ein Gerät oder Emulator
- Fertig! 🎉

## 🆘 Häufigste Probleme

**"Gradle sync failed"**
→ File → Invalidate Caches → Invalidate and Restart

**"SDK not found"**
→ File → Project Structure → SDK Location → Pfad prüfen

**"JDK Error"**
→ File → Project Structure → JDK 17 auswählen

**Alle anderen Probleme:**
→ Siehe **TROUBLESHOOTING.md**

## 🎯 Was macht diese App?

Die App zeigt alle Inhalte von **hundesportteam.de**:

✅ **Blog** - Alle Vereins-Blog-Beiträge  
✅ **Verein** - Alle Vereinsseiten  
✅ **Training** - Trainingsordnung & Unterseiten  
✅ **Offline** - Funktioniert auch ohne Internet  
✅ **Dark Mode** - Hell/Dunkel umschaltbar  

---

## 💡 Erster Start der App

Beim ersten App-Start:
1. Lädt die App Daten von der Website (5-10 Sekunden)
2. Speichert alles lokal für Offline-Nutzung
3. Zeigt die Blog-Beiträge an

Sie können dann zwischen den Tabs wechseln:
- **Blog** (Artikel-Icon)
- **Verein** (Haus-Icon)
- **Training** (Hantel-Icon)
